# SQL 优化器

> EXPLAIN 分析、索引建议、AI 优化建议

---

## 基本信息

- **规格编号**: `SPEC-014`
- **功能名称**: SQL 优化器
- **优先级**: P1 (重要功能)
- **状态**: ✅ 已完成

---

## 一、功能概述

### 1.1 核心功能

- SQL EXPLAIN 分析
- 索引建议
- AI 优化建议（豆包大模型）
- SQL 格式化

### 1.2 分析维度

| 维度 | 说明 |
|------|------|
| 执行计划 | EXPLAIN 结果可视化 |
| 索引使用 | 是否使用索引、索引效率 |
| 扫描行数 | 预估扫描行数 |
| 优化建议 | AI 生成优化建议 |

## 三、核心接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/v1/sql-optimizer/analyze` | POST | 分析 SQL |
| `/api/v1/sql-optimizer/explain` | POST | 执行 EXPLAIN |
| `/api/v1/sql-optimizer/suggest` | POST | AI 优化建议 |
