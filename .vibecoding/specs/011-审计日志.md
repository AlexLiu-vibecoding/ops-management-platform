# 审计日志

> 操作记录、查询导出

---

## 基本信息

- **规格编号**: `SPEC-011`
- **功能名称**: 审计日志
- **优先级**: P1 (重要功能)
- **状态**: ✅ 已完成

---

## 一、功能概述

### 1.1 记录的操作

| 类型 | 说明 |
|------|------|
| 登录/登出 | 认证相关 |
| 实例操作 | 创建/修改/删除实例 |
| 变更操作 | 提交/审批/执行变更 |
| 脚本执行 | 执行脚本 |
| 定时任务 | 任务操作 |
| 系统配置 | 配置修改 |

### 1.2 核心功能

- 日志查询
- 多维度筛选（用户/操作/时间）
- 日志导出
- 日志详情查看

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/v1/audit/logs` | GET | 日志列表 |

| `/api/v1/audit/export` | GET | 导出日志 |

---

## 四、日志字段

```python
class AuditLog:
    id: int
    user_id: int           # 操作用户
    username: str          # 用户名
    action: str            # 操作类型
    resource_type: str     # 资源类型
    resource_id: int       # 资源ID
    detail: dict           # 操作详情
    ip_address: str        # IP 地址
    user_agent: str        # 用户代理
    created_at: datetime   # 操作时间
```

---

## 五、注意事项

- 日志不可删除/修改
- 敏感信息脱敏处理
- 日志保留周期可配置
