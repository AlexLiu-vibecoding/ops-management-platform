# 监控中心

> 性能监控、慢查询分析、高 CPU SQL 监控、CloudWatch 集成

---

## 基本信息

- **规格编号**: `SPEC-006`
- **功能名称**: 监控中心
- **优先级**: P1 (重要功能)
- **状态**: ✅ 已完成

---

## 一、功能概述

### 1.1 监控类型

| 类型 | 说明 |
|------|------|
| 性能监控 | QPS、连接数、慢查询数等 |
| 慢查询监控 | 慢查询日志分析 |
| 高 CPU SQL | CPU 消耗高的 SQL |
| 主从复制 | 复制状态、延迟 |
| 事务与锁 | 锁等待、死锁 |

### 1.2 数据来源

| 来源 | 说明 |
|------|------|
| 本地实例 | 直接查询 performance_schema |
| AWS RDS | CloudWatch API |

---

## 二、实现位置

### 2.1 后端

| 文件 | 说明 |
|------|------|
| `backend/app/api/monitor.py` | 监控 API |
| `backend/app/api/monitor_ext.py` | 监控扩展（主从/锁） |
| `backend/app/api/alerts.py` | 告警 API |
| `backend/app/services/metrics.py` | 指标采集 |

### 2.2 前端

| 文件 | 说明 |
|------|------|
| `frontend/src/views/monitor/performance.vue` | 性能监控 |
| `frontend/src/views/monitor/slow-query.vue` | 慢查询监控 |
| `frontend/src/views/monitor/settings.vue` | 监控配置 |
| `frontend/src/views/alerts/index.vue` | 告警中心 |
| `frontend/src/views/replication/index.vue` | 主从复制 |
| `frontend/src/views/locks/index.vue` | 事务与锁 |

---

## 三、核心接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/v1/monitor/performance/{instance_id}` | GET | 性能指标 |
| `/api/v1/monitor/slow-queries/{instance_id}` | GET | 慢查询列表 |
| `/api/v1/monitor/replication/{instance_id}` | GET | 主从复制状态 |
| `/api/v1/monitor/locks/{instance_id}` | GET | 锁信息 |

---

## 四、告警规则

支持配置告警规则：

| 指标 | 条件 | 级别 |
|------|------|------|
| 慢查询数 | > 阈值 | warning |
| 连接数 | > 80% | warning |
| 复制延迟 | > 阈值 | critical |
| CPU 使用率 | > 阈值 | warning |

---

## 五、注意事项

- CloudWatch 指标采集需要 AWS 凭证
- 监控数据可配置保留周期
- 告警需要配置通知通道
