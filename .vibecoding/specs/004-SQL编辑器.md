# SQL 编辑器

> 语法高亮、执行、风险检测

---

## 基本信息

- **规格编号**: `SPEC-004`
- **功能名称**: SQL 编辑器
- **优先级**: P1 (重要功能)
- **状态**: ✅ 已完成

---

## 一、功能概述

### 1.1 核心功能

- SQL 语法高亮（CodeMirror）
- SQL 执行
- 结果展示（表格/JSON）
- 风险 SQL 检测
- 执行历史记录

### 1.2 支持的数据库

- MySQL
- PostgreSQL

## 三、核心接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/v1/sql-editor/execute` | POST | 执行 SQL |
| `/api/v1/sql-editor/validate` | POST | 校验 SQL |
| `/api/v1/sql-editor/history` | GET | 执行历史 |

---

## 四、风险检测

自动检测以下风险 SQL：

| 类型 | 示例 |
|------|------|
| 无 WHERE 的 UPDATE/DELETE | `DELETE FROM users` |
| DROP 操作 | `DROP TABLE xxx` |
| TRUNCATE 操作 | `TRUNCATE TABLE xxx` |
| 高影响行数 | 影响超过阈值 |

---

## 五、注意事项

- SELECT 默认限制返回行数
- 风险 SQL 需要二次确认
- 执行记录保存到审计日志
