# 实例管理

> MySQL/PostgreSQL/Redis 实例管理、AWS RDS 支持、连接测试

---

## 基本信息

- **规格编号**: `SPEC-002`
- **功能名称**: 实例管理
- **优先级**: P0 (核心功能)
- **状态**: ✅ 已完成

---

## 一、功能概述

### 1.1 支持的实例类型

| 类型 | 说明 |
|------|------|
| MySQL | 自建 MySQL / AWS RDS MySQL |
| PostgreSQL | 自建 PostgreSQL / AWS RDS PostgreSQL |
| Redis | 单机/集群/哨兵模式 |

### 1.2 核心功能

- 实例 CRUD 管理
- 连接测试
- 实例详情查看
- 关联环境管理
- AWS RDS CloudWatch 监控集成

## 三、核心接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/v1/instances` | GET | 实例列表 |
| `/api/v1/instances` | POST | 创建实例 |
| `/api/v1/instances/{id}` | GET | 实例详情 |
| `/api/v1/instances/{id}` | PUT | 更新实例 |
| `/api/v1/instances/{id}` | DELETE | 删除实例 |
| `/api/v1/instances/{id}/test` | POST | 测试连接 |

---

## 四、数据模型

```python
class Instance:
    id: int
    name: str              # 实例名称
    host: str              # 主机地址
    port: int              # 端口
    db_type: str           # mysql/postgresql/redis
    environment_id: int    # 关联环境
    is_enabled: bool       # 是否启用
    aws_region: str        # AWS 区域（RDS）
```

---

## 五、注意事项

- 敏感信息（密码）加密存储
- AWS RDS 实例需要配置区域凭证
- Redis 支持单机/集群/哨兵三种模式
