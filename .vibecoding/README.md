## 快速开始

### AI 开发流程

```
1. 开发前：读取 AGENTS.md
2. 复杂需求：使用 template.md 写规格
3. 开发中：对照 CheckList 检查
4. 完成后：让用户验收
```

### 新增功能步骤

```
□ 确认需求范围
□ 写规格（可选，复杂需求）
□ 配置数据库（菜单、权限）
□ 开发后端 API
□ 开发前端页面
□ 测试验证
□ 记录经验到 AGENTS.md
```
