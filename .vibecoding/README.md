## 快速开始

### AI 侧

```
1. 开发前：读取 AGENTS.md（唯一必读文档）
2. 复杂需求：填写 specs/template.md
3. 开发中：对照检查清单
4. 完成后：按验收模板确认
```

### 用户侧

```
- 开发开始时说："先看 AGENTS.md"
- 新功能说："请先写规格"
- 踩坑/新偏好说："记录到 AGENTS.md"
```
