## 快速使用

### AI 侧

```
1. 每次开发前：读取 .vibecoding/VIBECODING.md
2. 复杂需求：填写 specs/template.md
3. 开发过程中：对照 CHECKLIST.md
4. 开发完成后：按验收模板确认
5. 有新经验：更新 VIBECODING.md
```

### 用户侧

```
- 开发开始时说："先看 VIBECODING"
- 新功能说："请先写规格"
- 踩坑/新偏好说："记录到 VIBECODING"
```
